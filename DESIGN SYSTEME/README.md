# Design System — Ivoire Challenge Corporation (2C)

Langage visuel du site vitrine 2C. Deux formats du **même** système :

| Fichier | Rôle | Comment l'utiliser |
|---|---|---|
| [`design-system.md`](./design-system.md) | **Référence écrite** — spécifications, règles, tokens CSS/JSON | À lire / copier‑coller pour l'intégration ; source de vérité versionnée |
| [`styleguide.html`](./styleguide.html) | **Style guide visuel interactif** | Ouvrir dans un navigateur (double‑clic). Autonome : polices embarquées, aucune connexion requise |

## Socle de marque

- **Couleurs** — `#ffffff` (blanc) · `#08699b` (bleu 2C, primaire) · `#ed9f00` (or 2C, accent)
- **Typographies** — **DM Sans** (titres) · **Inter** (corps, UI, données)

## Règle d'accessibilité clé

L'or `#ed9f00` **ne passe pas** en texte sur blanc (contraste 2,20:1). Utiliser :
- `#a76700` pour du **texte or sur blanc** (4,58:1, conforme WCAG AA)
- texte foncé `#10222e` sur un **aplat or** (9,56:1, AAA)
